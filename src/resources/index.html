Holi, soy un servidor web super simple, tipo años 2000

Pretendo cumplir con los siguientes requisitos:

  • Escucha conexiones TCP en un puerto configurable mayor a 1024 y opera de forma continua.
  • Es multi-hilo, creando un hilo independiente por cada solicitud HTTP recibida.
  • Interpreta solicitudes HTTP/1.0 utilizando el método GET.
  • Lee y muestra por consola la línea de solicitud y los encabezados HTTP.
  • Responde con una estructura HTTP válida usando CRLF.
  • Sirve archivos HTML e imágenes JPG y GIF con su tipo MIME correcto.
  • Maneja recursos inexistentes devolviendo HTTP 404.
  • Cierra correctamente sockets y streams sin afectar solicitudes concurrentes.
